# VAPTSchema Builder Skill Resources

This directory contains resources used by the VAPTSchema Builder skill to process the VAPT-Risk-Catalogue-Full-125-v3.4.1.json dataset and build robust Interface Schema JSONs.

## Contents
* `schema-template.json` - A base template for the Interface Schema output.
* `VAPT-Risk-Catalogue-Full-125-v3.4.1.json` - The core risk catalog (referenced externally in the plugin `.data` directory).
